# 📝 Simple Todo App

A lightweight, browser-based Todo application built with pure JavaScript, HTML, and CSS. The app features persistent data storage so your tasks remain saved even if you refresh or close the page.

---

## 🚀 Features

*   **Create Tasks:** Easily add new tasks to your list.
*   **Persistent Storage:** Uses browser `localStorage` to save your tasks permanently.
*   **Delete Tasks:** Quickly remove individual items from your list via an intuitive delete icon.
*   **Validation:** Prevents empty inputs with an automatic alert system.

---

## 🛠️ Built With

*   **HTML5** - Structured user interface components.
*   **CSS3** - Clean styling and hover interactions for UI elements.
*   **JavaScript** - Core logic, DOM manipulation, and storage management.
*   **Font Awesome** - Modern, scalable vector icons for the user actions.
